## 项目简介（150字以内）

**项目名称：** 更懂你的智能支付推荐 —— 基于大小模型协同 + Skill工具库 + 个人知识库的AI支付推荐引擎

**项目简介：**  
传统支付推荐依赖规则配置，用户面临「选择过载」——支付工具太多、营销太复杂，系统只懂规则不懂用户。  
本方案通过**大模型理解意图 + 小模型精排决策 + Skill模块化能力 + 用户自然语言画像**四层架构，让支付推荐从「规则驱动」升级为「AI驱动」。  
核心创新：以「三力评分模型（惯性力+吸引力+约束力）」替代固定排序，实现**越用越准**的自学习闭环。

## 项目亮点

### 核心痛点（300字以内）

| 痛点 | 说明 | 影响 |
|------|------|------|
| **选择过载** | 用户在收银台面对白条、银行卡、小金库、先享后付等 10+ 支付选项 | 决策困难，转换率低 |
| **规则不透明** | 传统推荐按固定规则排序，无法解释为什么推这个 | 用户不信任系统 |
| **无个性化能力** | 规则配置迭代周期长（30天），冷启动慢，人力成本高 | 新用户无精准推荐 |
| **缺乏自学习** | 推荐效果不随用户使用行为提升，固定配置长期不变 | 越用越「笨」 |

### 关键创新点（300字以内）

1. **三力评分引擎**：创新设计「惯性力（F1）+ 吸引力（F2）+ 约束力（F3）」三元评分模型，替代传统固定排序，实现动态最优推荐
2. **自然语言用户画像**：支付回写用自然语言描述（非结构化字段），AI可直接理解，跨部门复用
3. **对话式反馈**：用户支付结果以「对话」形式回写（如「用户选择了白条+¥1.16优惠+3期服务费9折」），形成可读记忆
4. **大小模型协同**：大模型负责意图理解与补全，小模型负责精排与校验

### 价值贡献（成本/效率/体验/增长）

| 维度 | 贡献说明 | 预期 |
|------|---------|------|
| **成本** | 减少规则配置人力投入，Skill库即插即用 | 50%+ |
| **效率** | AI即时决策替代 30 天规则迭代周期 | 10× |
| **体验** | 用户不再困惑「为什么推这个」，推荐理由透明可追溯 | 全流程透传 |
| **增长** | 多卡用户转化率提升，潜客推荐准确率 80%+ | 40%+ |

### 方案介绍（500字以内）

**四层架构：**

```
┌──────────────────────────────────────┐
│        大模型（意图理解+补全）        │
│  读懂用户：场景+偏好+行为补全         │
├──────────────────────────────────────┤
│        小模型（精排+校验）           │
│  三力评分：惯性F1×0.45+吸引F2×0.35   │
├──────────────────────────────────────┤
│     Skill工具库（即插即用）          │
│  用户画像/画像漂移/推荐/回写/纠错     │

│  "用户两次用白条+¥1.16优惠+信誉卡备选" │
└──────────────────────────────────────┘
```

**五步自学习闭环：**  
① 画像生成 → ② 智能推荐 → ③ 支付回写 → ④ 知识沉淀 → ⑤ 能力迭代

## 评审口径补充（按5C原则）

### 1. 创新性（Creativity）
- **三力模型**：首次将「惯性力+吸引力+约束力」三元评分引入支付推荐（传统只有固定排序）
- **自然语言画像**：用户画像用自然语言描述（不是标签拼接），任何AI模型可直接理解
- **对话式回写**：支付结果不以字段写入，以「对话」形式回写，形成可读记忆

### 2. 聚焦性（Focus）
- **单点突破**：只聚焦「更懂用户的支付推荐」一个核心痛点
- **不贪大**：不做全支付平台，只做收银台推荐环节的精简引擎

### 3. 可行性（Feasibility）
- **已验证**：通过 6 个随机用户 pin 的历史数据验证，推荐准确率 **80%+**
- **数据完整**：617交易及历山数据 1,060+ 行、模型数据 1,100+ 行，覆盖多种用户类型
- **最快输出**：Skill 工具库在 2 天内可完成核心逻辑+4个银行通道接入

### 4. 价值贡献（Value）
- **成本**：减少规则配置周期 30 天 → AI即时 => **50%+**
- **效率**：一次签约多卡自动托管 => **10×**
- **体验**：用户不再「选哪个」→ 系统「自动最优」
- **增长**：多卡用户转化率提升 **40%+**，推荐准确率 **25%+**

### 5. 孵化潜力（孵化）
- **全京东生态**：可扩展至商城/金融/健康/物流全场景
- **A2P2基础能力**：可输出为第三方商家的支付推荐能力
- **端侧AI场景**：可扩展至手机AI助手（华为小艺/小米大模型）
